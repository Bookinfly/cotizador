Cotizador Freelance

Una guía para aprender a calcular tus presupuestos

Basado en mi experiencia como fotógrafo freelance, he desarrollado esta herramienta para ayudarte a calcular tus precios de forma rápida y sencilla. Aunque enfocada en fotografía, es útil para profesionales de cualquier área freelance que busquen optimizar su proceso de presupuestación.

Define la categoría de tu servicio

Seas especialista, generalista o novato, esta herramienta es útil incluso si solo realizas trabajos complementarios en fotografía.

Selecciona un nivel intermedio del servicio que ofreces. No estamos hablando de tu nivel general, sino de tu experiencia específica en esta área.

Nivel

¿Cuánto necesitas para tu estilo de vida?

Si este trabajo es parte de tus actividades principales o secundarias y afecta tu economía mensual, calcula cuánto necesitas ganar como mínimo cada mes. Algunos trabajos o clientes podrían no ser rentables en relación al tiempo y esfuerzo que requieren, dependiendo de tus metas económicas.

Si el trabajo es excepcional o para llenar huecos de agenda, puedes establecer un precio más bajo.

Esta aplicación web es solo de frontend: todo el código se ejecuta en tu navegador y no guarda ni envía tus datos.

Ganancia miníma mensual esperada

Si solo hicieras este tipo de servicio, ¿cuántos podrías realizar al mes?

Calcula cuántos servicios puedes ofrecer al mes, teniendo en cuenta la temporada y el tiempo que dedicas a tareas como ventas y publicidad.

Si tienes poca experiencia o equipo limitado, deberás hacer más producciones para compensar.

Recuerda considerar:
  • Tiempo de cobertura
  • Tiempo de preproducción
  • Tiempo de postproducción
  • Tiempo dedicado a marketing y publicidad para mantener el flujo de clientes
  • Tiempo invertido en ventas

Gastos

Todo negocio tiene gastos: vehículo, estudio, seguros e impuestos son algunos de los principales.

Algunos impuestos, como el IVA, se pueden agregar al precio final, pero otros, como ingresos brutos o el impuesto al vehículo, debes cubrirlos.

Identifica los gastos que debes amortizar, ya que no forman parte de tus ganancias.

Fondo para equipos

Los equipos se desgastan y pueden dañarse, por lo que es crucial estimar su valor para poder amortizarlos adecuadamente.

Este fondo tiene como objetivo cubrir reparaciones, reemplazos por desgaste o daño, actualizaciones de equipos y, en caso de robo, permitirte reemplazarlos.

Proyecto

Este proyecto es una herramienta práctica pensada tanto para el aprendizaje como para enriquecer mi portafolio. Además, puede ser útil para freelancers que enfrentan dificultades al realizar presupuestos, especialmente al inicio de su carrera. Diseñé este algoritmo para simplificar el proceso y lo integré en una interfaz amigable. Esta es solo la primera versión, pero planeo ampliarla y mejorarla próximamente.

Datos

El proyecto está desarrollado en frontend utilizando solo HTML, CSS y Vanilla JS. Está alojado en una Github Page, lo que significa que no utiliza cookies ni tiene capacidad para almacenar datos. Todo el procesamiento se realiza directamente en tu navegador.
